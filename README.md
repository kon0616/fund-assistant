# 💰 基金投资助手

一个基于 Streamlit 的个人基金持仓监控仪表盘，适合基金新手使用。

## ✨ 功能

| 标签页 | 功能 |
|--------|------|
| 📊 **今日看板** | 总资产、今日收益、翻倍进度条、各基金详情表格 |
| 📈 **持仓分析** | 涨跌幅柱状图、多周期热力图、净值走势、持仓饼图 |
| 📰 **每日情报** | 市场简报（≤200字）、行业新闻、基金术语学习 |

## 🚀 快速开始

### 1. 安装 Python（需要 3.9+）

从 [python.org](https://www.python.org/downloads/) 下载安装。

### 2. 安装依赖

```bash
cd "my_fund_assistant"
pip install -r requirements.txt
```

### 3. 启动仪表盘

```bash
streamlit run app.py
```

浏览器会自动打开 `http://localhost:8501`。

## ⚙️ 数据模式

| 模式 | 说明 |
|------|------|
| 📡 **模拟数据**（默认） | 离线可用，基于真实持仓模拟波动 |
| 🌐 **真实数据** | 通过 akshare 获取实时净值（需要网络） |

在侧边栏点击开关即可切换模式。

## 🤖 AI 简报（可选）

填入 **DeepSeek API Key** 后，每日情报页面的简报将由 AI 个性化生成（替代预设模板）：

1. 访问 https://platform.deepseek.com 注册并获取 API Key
2. 在侧边栏「AI 简报设置」输入 Key
3. 刷新页面，简报自动切换为 AI 生成

AI 会根据你的持仓数据：
- 分析今日涨跌背后的原因
- 识别表现最好/最差的基金并给出建议
- 筛选与持仓行业相关的最新动态
- 给新手一条个性化的投资建议

不填 Key 也不影响其他功能，系统会自动使用预设模板。
